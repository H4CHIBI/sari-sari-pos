## 📝 Project Overview
This system is built to automate the daily transactions of a Sari-Sari store. It focuses on simplifying sales recording and keeping an accurate count of stocks to prevent manual errors.

### 🚀 Key Features
* **Real-time Inventory:** Automatically deducts stocks when a sale is made.
* **Sales Reporting:** Track daily and monthly earnings.
* **User Authentication:** Secure login for owners and staff members.
* **Responsive Design:** Styled with **Tailwind CSS** for a clean, modern user interface.
* **Search Functionality:** Quickly find items by name or category.

## 🛠️ Tech Stack
* **Backend:** PHP (Native or Framework)
* **Frontend:** HTML5, Tailwind CSS
* **Database:** MySQL / MariaDB
* **Tools:** VS Code, XAMPP
